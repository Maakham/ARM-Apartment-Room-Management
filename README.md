# 🏢 ARM (Apartment Room Management)

ระบบบริหารจัดการอพาร์ทเมนต์และห้องเช่าแบบครบวงจร

## ✨ Features (ฟีเจอร์หลัก)
* จัดการสัญญาเช่า (Contract Management)
* คำนวณบิลค่าน้ำค่าไฟอัตโนมัติ (Utility Billing)

## 🚀 Installation (วิธีการติดตั้ง)
1. Clone repository
2. ติดตั้ง Python และรันคำสั่ง `pip install -r requirements.txt`
3. รันเซิร์ฟเวอร์ด้วยคำสั่ง `python manage.py runserver`
